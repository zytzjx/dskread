# README #

this is using sg3_lib.

### How To Compile sg3_lib ###

* Get from [SG3_LIB](http://sg.danny.cz/sg/p/sg3_utils-1.42.tar.xz)
* Version(1.4.2)

### How do I get set up? ###

* ./configure --prefix=/usr --disable-static &&
make
* sudo make install

### library ###

* Installed Library:libsgutils2.so
